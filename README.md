# Consyl Framework
This is a collection of code that can be used for making an ASCII Console games in C++.
## Requirements
- You need to have `g++` GNU C++ compiler installed.
- System must be on Windows or Linux
## How to Use
- Download source code
- Insert your code / gameplay logic (default level is SampleLevel)
- Run `run.bat` on Windows or `run.sh` on Linux

Caution: Untested on Linux!
